Resource Manager

Helping Google Workspace™ Admins manage Calendar Resources like Buildings, Resources, and Features directly from Google Sheets™.

Back to Add-ons

About

With Resource Manager, Google Workspace™ Admins can easily perform the following tasks without navigating the complex admin console:

  • List Buildings, Resources, and Features
  • Create new Buildings, Resources, and Features
  • Update existing Buildings and Resources
  • Delete Buildings and Resources in bulk
  • Add Features to Resources seamlessly

How to Install?

One-Time Setup

  1. Go to sheets.new and login with your corporate email id.
  2. From the menu above, click on Extensions (or Add-ons) > Get add-ons.
  3. Search for Resource Manager.
  4. Click on Individual Install / Domain Install > Click on Continue.
  5. Click on Allow to grant necessary permissions.

How to Use It?

Post-Installation Steps

  1. Re-load the same sheet where you have installed the Add-on.
  2. Go to Extensions > Resource Manager > Choose your desired action (e.g., List Buildings).
  3. The selected calendar resources or buildings will be populated in your sheet momentarily.

Note: You can reuse the exact same Google Sheet for future management tasks.
