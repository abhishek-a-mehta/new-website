How do I bulk suspend users based on a CSV file using GAMADV-XTD3?

You can achieve this by reading the CSV and passing the user identifiers to the suspend command. Ensure your CSV has a column header named `email`.

bash
gam csv users.csv gam update user ~email suspended true
